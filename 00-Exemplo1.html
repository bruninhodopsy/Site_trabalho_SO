Exemplo de Media Queries na Prática

Mesma página, comportamentos diferentes em telas diferentes.

O que são Media Queries?

Media Queries são regras especiais escritas dentro do arquivo CSS que dizem: “se a mídia (tela) tiver essas características, use estes estilos”.

Na prática, isso permite que a mesma página se comporte de maneiras diferentes em celulares, tablets e desktops, sem mudar o HTML.

Galeria Responsiva

Abaixo, uma galeria de imagens usando CSS Grid. Em telas grandes, temos 4 colunas. Em telas menores que 768px, a galeria passa a ter 2 colunas. Em telas bem pequenas (menores que 480px), vira 1 coluna. 

/* Estilos gerais (valem para todas as telas) */
.galeria {
  display: grid;
  gap: 16px;
  grid-template-columns: repeat(4, 1fr);
}

/* Estilos só para telas com até 768px de largura */
@media (max-width: 768px) {
  .galeria {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Estilos só para telas com até 480px de largura */
@media (max-width: 480px) {
  .galeria {
    grid-template-columns: 1fr;
  }
}
